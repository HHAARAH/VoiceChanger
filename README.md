## How to copy this project

# Clone it in bash
- git clone https://github.com/HHAARAH/VoiceChanger.git

# Import in IDE

# If "No such file or directory"
- Go to properties > C/C++ General > Paths and Symbols
- See if there is path "/Core/Inc", if no, add it manually or regenerate the code

# If "Device is not supported"
- Update the IDE and Install the needed package by clicking "Help"
